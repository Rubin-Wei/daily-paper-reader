## 摘要
机器人操作通常需要记忆：遮挡和状态变化可能导致决策时的观测产生感知混淆（perceptually aliased），使得动作选择在观测层面呈现非马尔可夫性，因为相同的观测可能源于不同的交互历史。大多数具身智能体通过语义压缩的轨迹和基于相似性的检索来实现记忆，这会丢弃用于消除歧义的细粒度感知线索，并可能返回感知上相似但与决策无关的情景。受人类情景记忆的启发，我们提出了 Chameleon，它通过写入基于几何的多模态标记（tokens）来保留消除歧义的上下文，并通过可微记忆栈实现目标导向的回忆。我们还引入了 Camo-Dataset，这是一个基于 UR5e 真实机器人的数据集，涵盖了感知混淆下的情景回忆、空间追踪和序列操作。在各项任务中，Chameleon 在感知易混淆的设置下，相比强基准模型一致地提高了决策可靠性和长程控制能力。

## Abstract
Robotic manipulation often requires memory: occlusion and state changes can make decision-time observations perceptually aliased, making action selection non-Markovian at the observation level because the same observation may arise from different interaction histories. Most embodied agents implement memory via semantically compressed traces and similarity-based retrieval, which discards disambiguating fine-grained perceptual cues and can return perceptually similar but decision-irrelevant episodes. Inspired by human episodic memory, we propose Chameleon, which writes geometry-grounded multimodal tokens to preserve disambiguating context and produces goal-directed recall through a differentiable memory stack. We also introduce Camo-Dataset, a real-robot UR5e dataset spanning episodic recall, spatial tracking, and sequential manipulation under perceptual aliasing. Across tasks, Chameleon consistently improves decision reliability and long-horizon control over strong baselines in perceptually confusable settings.

### 论文总结：Chameleon —— 用于长程机器人操作的情景记忆

#### 1. 核心问题与整体含义（研究动机和背景）
在长程机器人操作中，智能体经常面临**感知歧义（Perceptual Aliasing）**挑战。这意味着在决策的关键时刻，当前的视觉观测可能是非马尔可夫的（Non-Markovian）——即仅凭当前的图像无法判断正确动作，必须依赖历史交互。例如，在“三杯球戏”中，决策时的三个杯子外观完全一致，机器人必须记住球最初被放在哪以及杯子是如何交换的。

现有的具身智能记忆机制（如基于 LLM 的 RAG 或语义压缩轨迹）往往会丢失细粒度的几何线索，或仅基于视觉相似性检索，容易引入无关干扰。本文受人类**内嗅皮层-海马体-前额叶皮层（EC-HC-PFC）**记忆系统启发，旨在构建一种能够保留细粒度几何上下文并实现目标导向检索的可微记忆架构。

#### 2. 方法论：核心思想与关键技术
Chameleon 采用“感知 $\rightarrow$ 记忆 $\rightarrow$ 策略”的闭环架构，核心组件包括：

*   **几何锚定感知（Perception）**：
    *   利用固定视图和手眼相机双视图。
    *   **核心思想**：将机器人末端执行器（EE）作为几何锚点，通过投影将 3D 位姿转化为 2D 图像坐标。
    *   **技术细节**：使用对极几何偏置（Epipolar Bias）引导跨视图注意力，确保特征融合符合物理一致性，生成“几何接地”的多模态 Token。
*   **分层可微记忆栈（Memory）**：
    *   **情景记忆（Episodic Memory）**：使用选择性状态空间模型（SSM，类似 Mamba）维护多个具有不同“半衰期”的时间槽（Slots），实现模式分离（Pattern Separation）。
    *   **工作记忆（Working Memory）**：整合情景检索内容，用于即时控制。
    *   **HoloHead（全息头）**：引入“潜意识想象”目标，强制记忆状态 $h_t$ 预测未来 2D/3D 轨迹点，从而实现模式补全（Pattern Completion），确保记忆是目标导向的。
*   **整流流匹配策略（Policy）**：
    *   基于条件整流流匹配（Conditional Rectified Flow Matching）生成末端执行器的未来轨迹，相比单步预测更具稳定性。

#### 3. 实验设计
*   **数据集与场景**：发布了 **Camo-Dataset**，包含 120 条真实 UR5e 机器人演示数据。
*   **任务类别**：
    1.  **情景回忆**：清洗指定的盘子（三个盘子外观一致，需记住用户最初指认的那个）。
    2.  **空间追踪**：三杯球戏（追踪被遮挡且位置交换的目标）。
    3.  **顺序操作**：按顺序添加三种调料（需记住已完成的步骤）。
*   **对比基准（Baselines）**：
    *   **Diffusion Policy**（扩散策略）
    *   **Flow Matching**（流匹配）
    *   **ACT**（动作分块转换器）
*   **消融实验**：包括无记忆变体、基于相似性的记忆库（Memory Bank）、纯 Mamba 模型、无几何背侧流、无 HoloHead 目标等 5 种变体。

#### 4. 资源与算力
*   **硬件**：使用单张 **NVIDIA RTX 4090 GPU** 进行训练。
*   **训练时长**：每个任务训练 20,000 步。
*   **推理性能**：端到端推理延迟平均为 **82ms**，支持实时操作。
*   **参数量**：可训练参数约为 **49.89M**（不含冻结的 DINOv2 编码器），与基线模型（如 Diffusion Policy 的 328M）相比更为轻量。

#### 5. 实验数量与充分性
*   **实验规模**：每个任务类别进行了 **36 次独立真实机器人测试**，总计超过 100 次实机评估。
*   **充分性**：实验涵盖了决策成功率（DSR）、操作成功率（MSR）和整体成功率（SR），并引入了 Cohen’s kappa 系数来排除随机猜测的影响。
*   **客观性**：通过 UMAP 可视化展示了记忆状态 $h_t$ 在感知歧义下的聚类分离现象，从表征层面验证了方法的有效性。实验设计对比了当前最主流的具身智能算法，具有较强的说服力。

#### 6. 主要结论与发现
*   **记忆是消除歧义的关键**：在感知歧义场景下，传统反应式策略（如 Diffusion Policy）表现接近随机猜测，而 Chameleon 在情景回忆任务中达到了 100% 的决策成功率。
*   **目标导向检索优于相似性检索**：实验证明，简单的记忆库（Memory Bank）在感知相似但决策不同的场景下会失效，而通过 HoloHead 训练的预测性记忆能更准确地提取任务相关信息。
*   **几何引导的重要性**：移除背侧几何流会导致空间追踪任务性能大幅下降，证明了物理几何线索在记忆写入阶段的必要性。

#### 7. 优点（亮点）
*   **生物启发性强**：成功将神经科学中的 EC-HC-PFC 模型转化为可计算的工程架构。
*   **解决非马尔可夫问题**：针对机器人领域长期存在的感知歧义问题提供了系统性的解决方案。
*   **高效且实时**：在保持高性能的同时，参数量较小且推理速度快，适合部署在真实硬件上。
*   **高质量数据集**：Camo-Dataset 为后续研究机器人长程记忆提供了极具挑战性的基准。
