## 摘要
KV 缓存压缩对于高效的长上下文大语言模型（LLM）推理至关重要。减少每个键值对占用空间的方法（如量化和低秩分解）与减少缓存序列长度的方法是正交的。在序列长度维度上，现有方法涵盖了从纯逐出（选择保留哪些 KV 对）到合并（将相似的 KV 对组合成更少的对）的范围。这两者都仍然锚定在原始缓存条目上。我们提出了 KVSculpt，它走向了这一频谱的另一端：我们不再选择或组合原始对，而是在连续嵌入空间中优化一组更小的、不受约束的 KV 对，以保留每一层的注意力行为。键（Keys）通过 L-BFGS 进行优化，值（Values）则通过最小二乘法以闭式解求解，每隔几步交替进行。在此基础上，我们引入了自适应预算分配，利用低成本的试点压缩运行，根据每个组件的难度在各层和 KV 头之间重新分配压缩预算。在具有 2048 个 token 上下文的 Qwen2.5-1.5B-Instruct 上，在压缩率 r 为 {0.3, 0.5, 0.7} 的情况下，与 Select+Fit（基于注意力分数的逐出结合最小二乘值拟合）相比，KVSculpt 将 KL 散度降低了 3.5-4.1 倍。自适应分配在不增加额外推理成本的情况下，使 KL 散度进一步降低了 1.3 倍。分析表明，压缩难度高度不均匀：各层的试点均方误差（MSE）差异可达 100 倍，而单层内的两个 KV 头之间的差异可达 467 倍，这证明了细粒度的预算分配至关重要。

## Abstract
KV cache compression is critical for efficient long-context LLM inference. Approaches that reduce the per-pair footprint -- quantization and low-rank decomposition -- are orthogonal to those that reduce the sequence length of the cache. Along the sequence-length dimension, existing methods range from pure eviction -- selecting which KV pairs to keep -- to merging, which combines similar pairs into fewer ones. Both remain anchored to the original cache entries. We propose KVSculpt, which moves to the other end of this spectrum: instead of selecting or combining original pairs, we optimize a smaller set of unconstrained KV pairs in continuous embedding space to preserve each layer's attention behavior. Keys are optimized via L-BFGS and values are solved in closed form via least squares, alternating every few steps. On top of this, we introduce adaptive budget allocation, which uses a cheap pilot compression run to redistribute the compression budget across layers and KV heads based on per-component difficulty. On Qwen2.5-1.5B-Instruct with 2048-token contexts, KVSculpt reduces KL divergence by 3.5-4.1x compared to Select+Fit -- attention-score eviction with least-squares value fitting -- across compression ratios r in {0.3, 0.5, 0.7}. Adaptive allocation provides an additional 1.3x KL reduction at no extra inference cost. Analysis reveals that compression difficulty is highly non-uniform: per-layer pilot MSE varies by up to 100x across layers, and the two KV heads within a single layer can differ by up to 467x -- demonstrating that fine-grained budget allocation is essential.

### 1. 核心问题与研究动机
*   **核心问题**：在长文本大语言模型（LLM）推理中，KV Cache（键值缓存）随序列长度线性增长，成为内存瓶颈。
*   **研究动机**：现有的减少序列长度的方法主要分为“逐出”（Eviction，选择子集）和“合并”（Merging，加权组合）。作者认为这些方法都局限于原始缓存条目的离散空间。由于旋转位置编码（RoPE）已经将位置信息嵌入到向量中，压缩后的 KV 对并不一定要是原始对的子集，而可以是连续嵌入空间中任何能还原注意力行为的向量。

### 2. 方法论：KVSculpt
KVSculpt 将 KV 缓存压缩重新定义为一个**蒸馏问题**，旨在优化一组不受限的连续向量。
*   **核心思想**：给定原始 $N$ 个 KV 对，优化 $k$ 个新的 KV 对（$k < N$），使得在使用这 $k$ 个对进行注意力计算时，其输出尽可能接近原始输出。
*   **关键技术细节**：
    *   **损失函数**：包含两部分：1) 输出均方误差（Output MSE），确保注意力结果一致；2) Log-Sum-Exp (LSE) 匹配，确保注意力权重的分布质量，这对于后续 token 的合并至关重要。
    *   **交替优化策略**：
        *   **Key 优化**：使用 **L-BFGS** 算法（拟阶牛顿法）在非凸的 Softmax 景观中寻找最优 Key。
        *   **Value 求解**：在 Key 固定时，Value 的最优解可以通过**岭回归（最小二乘法）**以闭式解的形式快速求得。
    *   **合成未来查询（Synthetic Queries）**：由于压缩时不知道未来的 Query，作者通过对上下文内容向量进行去 RoPE 处理，并在未来位置重新应用 RoPE，构建合成 Query 来增强泛化性。
    *   **自适应预算分配**：通过一个简短的“试点运行（Pilot Run）”测量各层和各头（Head）的压缩难度（MSE），将更多的缓存配额分配给难以压缩的层/头。

*   **评估指标**：测量压缩后模型输出与原始模型输出之间的 **KL 散度**（越低越好）。
*   **对比基线（Benchmarks）**：

    *   **Joint Opt**：使用离散门控（Hard Concrete）进行联合优化。
*   **压缩率**：测试了 $r \in \{0.1, 0.3, 0.5, 0.7\}$ 等多种比例。

### 6. 主要结论与发现
*   **性能飞跃**：在相同压缩率下，KVSculpt 的 KL 散度比最强基线（Select+Fit）降低了 **3.5–4.1 倍**。
*   **连续空间的优势**：仅仅优化 Value 是不够的，让 Key 在连续空间自由移动是性能提升的关键。

### 7. 优点
*   **范式创新**：打破了“KV 缓存必须是原始 token 子集”的思维定式，将其转化为连续优化问题。
*   **数学严谨**：结合了 L-BFGS 的二阶优化能力和最小二乘法的解析解，效率高于纯一阶优化。
*   **细粒度分配**：引入了基于试点 MSE 的自适应分配，充分利用了模型内部结构的非均匀性。

*   **计算开销**：压缩过程（170s）远慢于简单的逐出算法，限制了其在实时对话中的应用。
*   **评估维度**：主要关注 KL 散度（模型输出的一致性），缺乏在下游任务（如 MMLU、摘要生成）上的直接准确率评测。

*   **层间依赖**：目前的逐层优化忽略了层与层之间的级联效应，导致长路径下的误差累积。
