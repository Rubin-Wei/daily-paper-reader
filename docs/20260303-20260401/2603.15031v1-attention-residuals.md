## 摘要
采用 PreNorm 的残差连接是现代大语言模型（LLM）的标准配置，但它们以固定的单位权重累加所有层的输出。这种均匀聚合导致隐藏状态随深度增加而出现不受控的增长，从而逐渐稀释了每一层的贡献。我们提出了注意力残差（Attention Residuals，简称 AttnRes），它将这种固定累加替换为对前序层输出的 softmax 注意力机制，使每一层能够利用学习到的、依赖于输入的权重，选择性地聚合早期的表示。为了解决在大规模模型训练中对所有前序层输出进行注意力计算所带来的内存和通信开销，我们引入了 Block AttnRes。它将层划分为块，并对块级表示进行注意力计算，在保留完整 AttnRes 大部分收益的同时减少了内存占用。结合基于缓存的流水线通信和两阶段计算策略，Block AttnRes 成为标准残差连接的实用即插即用替代方案，且开销极小。缩放法则（Scaling law）实验证实，这种改进在不同模型规模下是一致的，消融实验验证了依赖内容的深度选择所带来的益处。我们进一步将 AttnRes 集成到 Kimi Linear 架构中（总参数 48B / 激活参数 3B），并在 1.4T token 上进行预训练。实验表明，AttnRes 缓解了 PreNorm 的稀释效应，使不同深度的输出幅度和梯度分布更加均匀，并提升了所有评估任务的下游性能。

## Abstract
Residual connections with PreNorm are standard in modern LLMs, yet they accumulate all layer outputs with fixed unit weights. This uniform aggregation causes uncontrolled hidden-state growth with depth, progressively diluting each layer's contribution. We propose Attention Residuals (AttnRes), which replaces this fixed accumulation with softmax attention over preceding layer outputs, allowing each layer to selectively aggregate earlier representations with learned, input-dependent weights. To address the memory and communication overhead of attending over all preceding layer outputs for large-scale model training, we introduce Block AttnRes, which partitions layers into blocks and attends over block-level representations, reducing the memory footprint while preserving most of the gains of full AttnRes. Combined with cache-based pipeline communication and a two-phase computation strategy, Block AttnRes becomes a practical drop-in replacement for standard residual connections with minimal overhead.   Scaling law experiments confirm that the improvement is consistent across model sizes, and ablations validate the benefit of content-dependent depth-wise selection. We further integrate AttnRes into the Kimi Linear architecture (48B total / 3B activated parameters) and pre-train on 1.4T tokens, where AttnRes mitigates PreNorm dilution, yielding more uniform output magnitudes and gradient distribution across depth, and improves downstream performance across all evaluated tasks.

# 《Attention Residuals》论文深度总结

### 1. 核心问题与整体含义（研究动机和背景）
现代大语言模型（LLM）普遍采用 **PreNorm 残差连接**（$h_l = h_{l-1} + f_{l-1}(h_{l-1})$），其本质是所有前序层输出的等权重累加。这种机制存在三个核心缺陷：
*   **PreNorm 稀释效应**：隐藏状态的幅度随深度 $O(L)$ 增长，导致深层网络中每一层的相对贡献被不断稀释，迫使深层学习更大的输出幅度以维持影响力。
*   **缺乏选择性检索**：每一层只能接收前一层的压缩状态，无法根据当前内容选择性地提取早期层的特定信息。
*   **不可逆的信息损失**：一旦信息在聚合中被淹没，深层网络难以重新找回。

本文受序列建模中“循环（RNN）向注意力（Transformer）演进”的启发，提出将**深度维度的固定累加**替换为**学习到的 Softmax 注意力机制**，即 **Attention Residuals (AttnRes)**。

### 2. 方法论：核心思想与关键技术
#### 核心思想
AttnRes 将残差连接视为一种“深度维度的注意力”。它不再简单求和，而是通过一个学习到的伪查询（pseudo-query）向量 $w_l$，对所有前序层的输出进行 Softmax 加权聚合。

#### 关键技术细节
*   **Full AttnRes（完整版）**：
    *   公式：$h_l = \sum_{i=0}^{l-1} \alpha_{i \to l} \cdot v_i$，其中 $v_i$ 是第 $i$ 层的输出。
    *   权重计算：使用 $\text{Softmax}$ 作用于查询 $w_l$ 与各层输出（经过 RMSNorm）的点积。
    *   特点：实现了内容相关的深度选择，但在大规模训练中存在 $O(Ld)$ 的内存和流水线并行通信开销。
*   **Block AttnRes（分块版）**：
    *   为了工程落地，将 $L$ 层划分为 $N$ 个块（通常 $N \approx 8$）。
    *   块内维持标准残差累加，块间进行注意力聚合。
    *   优势：将通信和内存开销从 $O(Ld)$ 降低到 $O(Nd)$，成为实用的即插即用模块。
*   **基础设施优化**：
    *   **跨阶段缓存（Cross-stage Caching）**：在流水线并行训练中，仅传输增量块表示，消除冗余通信。
    *   **两阶段推理策略**：利用伪查询与输入无关的特性，预先批处理块间注意力计算，将推理延迟增量控制在 2% 以内。

*   **理论优雅**：建立了“时间维度注意力”与“深度维度注意力”的对偶关系，完善了残差连接的理论框架。

*   **即插即用**：几乎不增加参数量（仅每层一个 $d$ 维向量），且对推理延迟影响极小。
