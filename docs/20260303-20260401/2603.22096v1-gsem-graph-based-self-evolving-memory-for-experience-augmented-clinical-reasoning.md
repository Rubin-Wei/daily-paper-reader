## 摘要
临床决策智能体可以从重用先前的决策经验中获益。然而，许多记忆增强方法将经验存储为独立的记录，缺乏明确的关系结构，这可能导致检索噪声、重用不可靠，在某些情况下甚至比直接的大语言模型（LLM）推理性能更差。我们提出了 GSEM（基于图的自演化记忆），这是一个临床记忆框架，它将临床经验组织成双层记忆图，既捕捉了每个经验内部的决策结构，也捕捉了经验之间的关系依赖，并支持适用性感知检索以及基于在线反馈的节点质量和边权重校准。在 MedR-Bench 和 MedAgentsBench 两个基准测试中，结合两种 LLM 主干网络，GSEM 在所有基准方法中实现了最高的平均准确率，在使用 DeepSeek-V3.2 和 Qwen3.5-35B 时分别达到了 70.90% 和 69.24%。代码可在 https://github.com/xhan1022/gsem 获取。

## Abstract
Clinical decision-making agents can benefit from reusing prior decision experience. However, many memory-augmented methods store experiences as independent records without explicit relational structure, which may introduce noisy retrieval, unreliable reuse, and in some cases even hurt performance compared to direct LLM inference. We propose GSEM (Graph-based Self-Evolving Memory), a clinical memory framework that organizes clinical experiences into a dual-layer memory graph, capturing both the decision structure within each experience and the relational dependencies across experiences, and supporting applicability-aware retrieval and online feedback-driven calibration of node quality and edge weights. Across MedR-Bench and MedAgentsBench with two LLM backbones, GSEM achieves the highest average accuracy among all baselines, reaching 70.90\% and 69.24\% with DeepSeek-V3.2 and Qwen3.5-35B, respectively. Code is available at https://github.com/xhan1022/gsem.

#### 1. 核心问题与整体含义（研究动机和背景）
大语言模型（LLM）在医疗决策中表现出色，但在处理高风险、信息不全且具有强条件依赖的真实临床病例时，仅依赖参数化知识往往不足。现有的记忆增强方法（如 RAG）通常将临床经验存储为独立的、无关联的记录，这导致了两个主要挑战：
*   **边界失效（Boundary Failure）**：检索到的经验仅部分匹配条件，忽略了关键的边界限制（如特定风险因素）。
*   **协作失效（Collaboration Failure）**：检索到的多个经验之间存在冲突或缺乏共同使用的价值，导致推理逻辑不连贯。
**GSEM** 旨在通过构建一个结构化的、可自演化的双层记忆图来解决这些问题，使智能体能够可靠地重用过去的成功经验并规避失败教训。

#### 2. 方法论：核心思想与关键技术
GSEM 提出了一个三阶段的临床记忆框架：

*   **双层记忆图构建**：
    *   **实体层（Gent）**：将每个经验分解为具有语义角色（条件、约束、行动、理由、结果）的实体节点，捕捉经验内部的决策流。

*   **适用性感知检索**：
    *   **混合种子召回**：结合基于实体的稀疏检索和基于嵌入的稠密检索，生成初始种子集。
    *   **多种子图遍历**：LLM 引导智能体在经验层图上执行离散动作（收集、探索、回溯、停止），根据节点质量 $Q$ 和边权重 $W$ 的综合得分选择路径，确保检索到的经验组合具有连贯性。
*   **反馈驱动的自演化**：
    *   任务完成后，根据结果反馈（标量信号 $\Delta t$）更新激活节点的质量 $Q$ 和相关边的权重 $W$。
    *   采用指数衰减的信用分配机制，强化对成功有贡献的经验关联，削弱导致失败的关联，实现非参数化的持续学习。
