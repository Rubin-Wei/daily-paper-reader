## 摘要
零样本目标导航要求智能体在没有先验地图或特定任务训练的情况下，在陌生环境中定位未见过的目标物体，这仍然是一个重大挑战。尽管视觉语言模型（VLM）的最新进展为该任务提供了极具前景的常识推理能力，但这些模型仍面临空间幻觉、局部探索死锁以及高层语义意图与底层控制之间脱节的问题。对此，我们提出了一种名为 ReMemNav 的新型分层导航框架，该框架将全景语义先验和情节记忆与 VLM 无缝集成。我们引入了 Recognize Anything Model 来锚定 VLM 的空间推理过程。我们还设计了一种基于情节语义缓冲队列的自适应双模态反思机制。该机制利用历史记忆主动验证目标可见性并修正决策，以防止死锁。在底层动作执行方面，ReMemNav 利用深度掩码提取一系列可行操作，允许 VLM 选择最优动作并映射为实际的空间移动。在 HM3D 和 MP3D 上的广泛评估表明，ReMemNav 在成功率和探索效率方面均优于现有的免训练零样本基准模型。具体而言，我们实现了显著的绝对性能提升，在 HM3D v0.1 上 SR 和 SPL 分别增长了 1.7% 和 7.0%，在 HM3D v0.2 上增长了 18.2% 和 11.1%，在 MP3D 上增长了 8.7% 和 7.9%。

## Abstract
Zero-shot object navigation requires agents to locate unseen target objects in unfamiliar environments without prior maps or task-specific training which remains a significant challenge. Although recent advancements in vision-language models(VLMs) provide promising commonsense reasoning capabilities for this task, these models still suffer from spatial hallucinations, local exploration deadlocks, and a disconnect between high-level semantic intent and low-level control. In this regard, we propose a novel hierarchical navigation framework named ReMemNav, which seamlessly integrates panoramic semantic priors and episodic memory with VLMs. We introduce the Recognize Anything Model to anchor the spatial reasoning process of the VLM. We also design an adaptive dual-modal rethinking mechanism based on an episodic semantic buffer queue. The proposed mechanism actively verifies target visibility and corrects decisions using historical memory to prevent deadlocks. For low-level action execution, ReMemNav extracts a sequence of feasible actions using depth masks, allowing the VLM to select the optimal action for mapping into actual spatial movement. Extensive evaluations on HM3D and MP3D demonstrate that ReMemNav outperforms existing training-free zero-shot baselines in both success rate and exploration efficiency. Specifically, we achieve significant absolute performance improvements, with SR and SPL increasing by 1.7% and 7.0% on HM3D v0.1, 18.2% and 11.1% on HM3D v0.2, and 8.7% and 7.9% on MP3D.

### 1. 核心问题与整体含义（研究动机和背景）
论文聚焦于**零样本目标导航（Zero-Shot Object Navigation, ZSON）**，即要求机器人在完全陌生的环境中，在没有先验地图或针对性训练的情况下，寻找并导航至指定类别的物体。
*   **研究动机**：虽然视觉语言模型（VLM）为导航提供了强大的常识推理能力，但现有的免训练（Training-free）方法存在三大痛点：
    1.  **空间幻觉**：VLM 容易根据局部纹理误判不存在的物体。
    2.  **局部探索死锁**：由于缺乏记忆，智能体常在局部区域（如死胡同）循环徘徊。
    3.  **语义与控制脱节**：高层语义指令难以精准转化为底层的避障与移动动作。

### 2. 方法论：核心思想与关键技术
ReMemNav 提出了一种分层认知框架，核心思想是**“感知锚定+记忆增强+双模态反思”**：
*   **语义视觉锚定（RAM）**：引入 Recognize Anything Model (RAM) 作为轻量级感知前端。它为全景图像提供结构化的标签字典，作为 VLM 的底层物理锚点，减少 VLM 直接处理原始图像时的幻觉。
*   **情节记忆缓冲队列（EMBQ）**：建立一个滑动窗口队列（容量 $K=10$），存储历史位置坐标和 VLM 生成的环境描述。这是一种比度量地图更轻量级的拓扑语义记忆。
*   **自适应双模态反思机制（ADRM）**：
    *   **模式 A（决策评估）**：当未发现目标时，系统通过坐标投影检查预选方向是否与历史路径重叠。若重叠，则触发反思，强制 VLM 结合历史描述选择新路径，打破死锁。
    *   **模式 B（目标验证）**：当 VLM 声称发现目标时，系统强制其对特定视角进行二次细致检查，过滤视觉伪影。
*   **底层动作执行**：利用深度掩码采样候选动作射线，并将其叠加在图像上，由 VLM 选择最优射线并映射为物理空间的极坐标移动。

### 6. 主要结论与发现
*   **性能领先**：ReMemNav 在所有免训练方法中表现最优。在 HM3D v0.2 上，SR 提升了 18.2%，SPL 提升了 11.1%；在 MP3D 上 SR 提升了 8.7%。
*   **效率提升**：在 HM3D v0.1 上，其 SPL（36.8%）甚至超过了需要大量训练的 SGM 方法，证明了反思机制能显著减少冗余探索。
*   **记忆容量悖论**：实验发现记忆容量并非越大越好，$K=10$ 为最优。过长的历史信息会引入上下文噪声，导致 VLM 注意力分散（信息过载）。

### 7. 优点与亮点
*   **双系统认知模型**：借鉴人类心理学的“快思考与慢思考”，通过反思机制实现了逻辑回溯，这在纯反应式导航中是一个突破。
*   **轻量化设计**：无需构建复杂的全局度量地图，仅靠语义标签和简短描述即可实现高效导航，降低了计算开销。
*   **即插即用**：框架具有模型无关性（Model-agnostic），可以随 VLM 能力的提升而持续进化。

### 8. 不足与局限
*   **VLM 依赖性**：性能高度依赖于底层 VLM 的推理能力（如 Gemini 与 Qwen 之间存在显著性能差距）。
*   **长程任务挑战**：目前的滑动窗口记忆可能无法处理极大规模、跨楼层的超长程导航任务，历史信息可能会被过早丢弃。
*   **实时性压力**：虽然免训练，但每一步都需要调用 VLM 进行多视角分析和反思，在实际机器人硬件上可能会面临推理延迟挑战。
