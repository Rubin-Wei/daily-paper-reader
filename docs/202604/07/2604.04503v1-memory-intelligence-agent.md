## 摘要
深度研究智能体（DRA）将大语言模型（LLM）推理与外部工具相结合。记忆系统使 DRA 能够利用历史经验，这对于高效推理和自主演化至关重要。现有方法依赖于从记忆中检索相似轨迹以辅助推理，但面临记忆演化效率低下以及存储和检索成本增加的关键局限。为了解决这些问题，我们提出了一种新型的记忆智能体（MIA）框架，由“管理者-规划者-执行者”（Manager-Planner-Executor）架构组成。记忆管理者是一个非参数化记忆系统，可以存储压缩后的历史搜索轨迹。规划者是一个参数化记忆智能体，能够为问题生成搜索计划。执行者是另一个智能体，可以在搜索计划的指导下搜索并分析信息。为了构建 MIA 框架，我们首先采用交替强化学习范式来增强规划者与执行者之间的协作。此外，我们使规划者能够在测试时学习过程中持续演化，在不中断推理过程的情况下，随推理同步进行实时更新。此外，我们在参数化记忆与非参数化记忆之间建立了双向转换循环，以实现高效的记忆演化。最后，我们引入了反思和无监督判断机制，以提升在开放世界中的推理和自我演化能力。在 11 个基准测试上的广泛实验证明了 MIA 的优越性。

## Abstract
Deep research agents (DRAs) integrate LLM reasoning with external tools. Memory systems enable DRAs to leverage historical experiences, which are essential for efficient reasoning and autonomous evolution. Existing methods rely on retrieving similar trajectories from memory to aid reasoning, while suffering from key limitations of ineffective memory evolution and increasing storage and retrieval costs. To address these problems, we propose a novel Memory Intelligence Agent (MIA) framework, consisting of a Manager-Planner-Executor architecture. Memory Manager is a non-parametric memory system that can store compressed historical search trajectories. Planner is a parametric memory agent that can produce search plans for questions. Executor is another agent that can search and analyze information guided by the search plan. To build the MIA framework, we first adopt an alternating reinforcement learning paradigm to enhance cooperation between the Planner and the Executor. Furthermore, we enable the Planner to continuously evolve during test-time learning, with updates performed on-the-fly alongside inference without interrupting the reasoning process. Additionally, we establish a bidirectional conversion loop between parametric and non-parametric memories to achieve efficient memory evolution. Finally, we incorporate a reflection and an unsupervised judgment mechanisms to boost reasoning and self-evolution in the open world. Extensive experiments across eleven benchmarks demonstrate the superiority of MIA.

### 1. 核心问题与整体含义（研究动机和背景）
深度研究智能体（Deep Research Agents, DRAs）通过结合 LLM 推理与外部工具（如搜索引擎）来处理复杂任务。然而，现有的智能体记忆系统存在三大痛点：
*   **效率低下**：长上下文记忆（Long-context memory）容易稀释注意力，引入噪声，且随着交互轮数增加，存储和检索成本呈指数级增长。
*   **缺乏过程导向**：现有记忆多侧重于“是什么”（事实知识），而深度研究更需要“怎么做”（搜索策略、失败教训、推理路径）。
*   **演化停滞**：多数系统在推理阶段是静态的，无法在开放世界中通过自我尝试实现自主进化。
**MIA 的核心目标**是构建一个具备“大脑”功能的记忆框架，通过解耦规划与执行，实现记忆的高效压缩与参数化内化。

### 2. 方法论：核心思想与关键技术
MIA 提出了 **Manager-Planner-Executor（管理者-规划者-执行者）** 架构：
*   **Memory Manager（非参数化记忆）**：模拟海马体，将冗长的搜索轨迹压缩为结构化的“工作流摘要”（Workflow Summaries），并根据语义相似度、价值奖励和频率奖励进行混合检索。
*   **Planner（参数化记忆智能体）**：负责生成搜索计划。通过训练，它能将历史经验内化为模型参数，减少对外部长上下文的依赖。
*   **Executor（执行智能体）**：负责具体工具调用（ReAct 模式）。
*   **关键技术细节**：
    *   **交替强化学习（Alternating RL）**：采用 GRPO 算法。第一阶段训练执行者遵循计划；第二阶段训练规划者吸收记忆并进行反思（Reflect-Replan）。
    *   **测试时学习（Test-Time Learning, TTL）**：在推理过程中同步更新规划者参数，实现“边做边学”。
    *   **无监督自我演化**：引入类似学术评审的“审稿人-领域主席”机制（逻辑、事实、有效性三个维度），在没有标准答案的情况下评估轨迹质量，驱动自我进化。

*   **实验规模**：论文进行了跨模态、跨领域的广泛测试，涵盖了从基础事实问答到复杂多跳推理的任务。

*   **客观性**：对比了多种 SOTA 记忆框架，并验证了 MIA 在更换不同闭源执行者（如 GPT-5.4）时的泛化性。实验设计考虑了“无监督”这一极具挑战性的真实场景，具有较高的客观性和前瞻性。

### 6. 主要结论与发现
*   **性能飞跃**：MIA 显著提升了现有最强模型的表现。例如，将 GPT-5.4 在 LiveVQA 上的表现提升了 9%。
*   **以小博大**：搭载 MIA 的 **Qwen2.5-VL-7B** 平均表现提升 31%，甚至超越了比它大得多的 Qwen2.5-VL-32B（领先 18%）。

*   **持续进化**：在无监督环境下，MIA 通过多次迭代（Epoch 1-3），性能稳步提升，证明了其自主学习能力。

### 7. 优点与亮点
*   **架构创新**：解耦了“规划”与“执行”，并实现了参数化与非参数化记忆的双向转换，兼顾了存储效率与推理深度。
*   **TTL 机制**：实现了真正的在线演化，无需中断推理即可更新模型策略。
*   **无监督评估**：提出的“类同行评审”评估框架为智能体在缺乏标签的开放世界中进化提供了切实可行的路径。
*   **工具效率**：分析显示 MIA 能通过更精准的工具调用（而非盲目增加调用次数）来解决问题。

### 8. 不足与局限
*   **算力开销**：虽然推理时记忆检索高效，但 TTL 阶段需要多次 rollout 和实时参数更新，这在实时性要求极高的场景下可能存在延迟挑战。
*   **模型依赖**：框架的效果高度依赖于作为“评判者”和“管理者”的高阶模型（如 Qwen3-32B）的质量。
*   **应用范围**：目前主要集中在“深度研究”（搜索与问答）任务，对于需要极长时序操作（如连续控制、复杂游戏）的泛化性尚待验证。
