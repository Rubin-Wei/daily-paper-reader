## 摘要
以人格为中心的记忆领域的最新进展揭示了多智能体系统在管理人格记忆方面的强大能力，特别是在对话场景中。然而，这些复杂的框架通常面临信息丢失的问题，且在不同场景下表现脆弱，导致性能欠佳。在本文中，我们提出了 DeltaMem，这是一种代理式记忆管理系统，它将以人格为中心的记忆管理建模为单智能体设置下的端到端任务。为了进一步提高代理式记忆管理器的性能，我们从人类记忆的演化中汲取灵感，合成了一个用户-助手对话数据集以及相应的操作级记忆更新标签。在此基础上，我们引入了一种新颖的基于记忆的编辑距离（Levenshtein Distance）来形式化记忆更新奖励，并提出了一种定制的强化学习框架，以进一步增强 DeltaMem 的管理能力。大量实验表明，无需训练的 DeltaMem 和经过强化学习训练的 DeltaMem 在包括 LoCoMo、HaluMem 和 PersonaMem 在内的多种长期记忆基准测试中，均优于所有产品级基准模型。

## Abstract
Recent advances in persona-centric memory have revealed the powerful capability of multi-agent systems in managing persona memory, especially in conversational scenarios. However, these complex frameworks often suffer from information loss and are fragile across varying scenarios, resulting in suboptimal performance. In this paper, we propose DeltaMem, an agentic memory management system that formulates persona-centric memory management as an end-to-end task within a single-agent setting. To further improve the performance of our agentic memory manager, we draw inspiration from the evolution of human memory and synthesize a user-assistant dialogue dataset along with corresponding operation-level memory updating labels. Building on this, we introduce a novel Memory-based Levenshtein Distance to formalize the memory updating reward, and propose a tailored reinforcement learning framework to further enhance the management capabilities of DeltaMem. Extensive experiments show that both training-free and RL-trained DeltaMem outperform all product-level baselines across diverse long-term memory benchmarks, including LoCoMo, HaluMem, and PersonaMem.

### 2. 方法论：DeltaMem 框架
*   **核心思想**：将记忆管理重新表述为**单智能体端到端任务**。利用 ReAct（Reasoning-and-Act）范式，让智能体自主决定检索什么以及如何更新记忆库。
*   **关键技术细节**：
    *   **单智能体 ReAct 模式**：DeltaMem 在处理新对话时，通过推理调用搜索工具，直接输出一系列记忆操作（Add, Update, None），减少了模块化带来的损耗。
    *   **记忆演化引导的数据合成**：受人类记忆演化启发，合成包含 25 个连续事件的对话流。涵盖“事件内演化”（属性扩展、调整、偏移）和“跨事件演化”（情感更新、意义重构）。
    *   **基于记忆的编辑距离（Memory-based Levenshtein Distance）**：为了解决操作级评估难题，将评估从“操作”转向“记忆状态”。通过计算预测状态与目标状态之间的语义对齐（Optimal Transport）和局部词汇保真度（关键字覆盖率），量化状态转换成本。
    *   **定制化 RL 框架**：采用 GRPO 算法，将上述编辑距离转化为软 F1 分数作为奖励信号，结合格式奖励和检索奖励进行策略优化。

*   **数据集/基准测试**：
    *   **LoCoMo**：评估极长对话（约 300 轮）中的事实召回、多跳推理和时间依赖。
    *   **HaluMem**：评估记忆系统中的幻觉行为，包括提取、更新和问答阶段。
    *   **PersonaMem**：评估动态用户画像跟踪和个性化响应生成。
*   **对比方法**：

### 6. 主要结论与发现
*   **单智能体更优**：即使是不经训练的 DeltaMem，在多个指标上也优于复杂的多智能体系统，证明了端到端推理能有效缓解信息丢失。
*   **RL 提升显著**：经过定制化 RL 训练后，DeltaMem-8B-RL 在 LoCoMo 上取得了 75.13 的高分，在生成新想法（New-Ideas）等任务上比非 RL 版本提升了 62%。
*   **稳定性强**：随着对话轮数增加，DeltaMem 的准确率下降曲线比 LightMem 等基准更平缓，表现出更好的长期维护能力。

### 7. 优点与亮点
*   **架构简化**：打破了传统的“提取-检索-更新”三段式教条，回归单智能体端到端控制。
*   **奖励设计新颖**：提出的“记忆编辑距离”巧妙地解决了自然语言记忆条目之间“语义相同但表述不同”的对齐问题，为记忆系统训练提供了非稀疏的监督信号。
*   **数据合成质量高**：模拟了人类记忆的动态演化逻辑，而非简单的随机事实堆砌。

### 8. 不足与局限
*   **合成数据偏差**：训练完全依赖于合成数据，虽然使用了 PersonaHub 增加多样性，但仍可能存在 LLM 固有的社会偏见或与真实用户行为的偏差。
*   **检索压力**：虽然 RL 减轻了检索负担，但在极大规模记忆库（百万级条目）下的扩展性仍需进一步验证。
*   **应用限制**：目前主要针对文本对话，尚未涉及多模态记忆或更复杂的结构化知识图谱管理。
