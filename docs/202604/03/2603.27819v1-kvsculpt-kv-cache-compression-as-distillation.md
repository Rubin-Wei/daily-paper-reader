## 摘要
KV 缓存压缩对于高效的长上下文大语言模型（LLM）推理至关重要。减少单对占用空间的方法（如量化和低秩分解）与减少缓存序列长度的方法是正交的。在序列长度维度上，现有方法涵盖了从纯逐出（选择保留哪些 KV 对）到合并（将相似的 KV 对组合成更少的对）的范围。这两类方法都受限于原始缓存条目。我们提出了 KVSculpt，它走向了这一频谱的另一端：我们不再选择或组合原始 KV 对，而是在连续嵌入空间中优化一组更小的、不受约束的 KV 对，以保留每一层的注意力行为。Key 通过 L-BFGS 进行优化，Value 则通过最小二乘法以闭式解形式求解，每隔几步交替进行。在此基础上，我们引入了自适应预算分配，利用低成本的初步压缩运行，根据每个组件的难度在各层和 KV 头之间重新分配压缩预算。在具有 2048-token 上下文的 Qwen2.5-1.5B-Instruct 上，在压缩率 r ∈ {0.3, 0.5, 0.7} 的情况下，与 Select+Fit（基于注意力分数的逐出结合最小二乘 Value 拟合）相比，KVSculpt 将 KL 散度降低了 3.5-4.1 倍。自适应分配在不增加额外推理成本的情况下，使 KL 散度进一步降低了 1.3 倍。分析表明，压缩难度高度不均匀：各层的初步均方误差（MSE）差异可达 100 倍，而单层内的两个 KV 头之间的差异甚至可达 467 倍，这证明了细粒度预算分配的必要性。

## Abstract
KV cache compression is critical for efficient long-context LLM inference. Approaches that reduce the per-pair footprint -- quantization and low-rank decomposition -- are orthogonal to those that reduce the sequence length of the cache. Along the sequence-length dimension, existing methods range from pure eviction -- selecting which KV pairs to keep -- to merging, which combines similar pairs into fewer ones. Both remain anchored to the original cache entries. We propose KVSculpt, which moves to the other end of this spectrum: instead of selecting or combining original pairs, we optimize a smaller set of unconstrained KV pairs in continuous embedding space to preserve each layer's attention behavior. Keys are optimized via L-BFGS and values are solved in closed form via least squares, alternating every few steps. On top of this, we introduce adaptive budget allocation, which uses a cheap pilot compression run to redistribute the compression budget across layers and KV heads based on per-component difficulty. On Qwen2.5-1.5B-Instruct with 2048-token contexts, KVSculpt reduces KL divergence by 3.5-4.1x compared to Select+Fit -- attention-score eviction with least-squares value fitting -- across compression ratios r in {0.3, 0.5, 0.7}. Adaptive allocation provides an additional 1.3x KL reduction at no extra inference cost. Analysis reveals that compression difficulty is highly non-uniform: per-layer pilot MSE varies by up to 100x across layers, and the two KV heads within a single layer can differ by up to 467x -- demonstrating that fine-grained budget allocation is essential.

### 1. 核心问题与研究动机
*   **核心问题**：在长上下文大语言模型（LLM）推理中，KV Cache（键值缓存）随序列长度线性增长，成为内存瓶颈。
*   **研究背景**：现有的减少序列长度的方法主要分为“逐出”（Eviction，选择子集）和“合并”（Merging，加权组合）。作者认为这些方法都受限于原始缓存条目的离散位置，限制了压缩后的表达能力。
*   **核心动机**：由于 RoPE 编码已将位置信息嵌入向量，KV 对在嵌入空间中实际上是连续且无序的。因此，压缩不应局限于选择原始条目，而应在连续空间中寻找能最完美模拟原始注意力行为的“虚拟”KV 对。

### 2. 方法论：KVSculpt 核心思想与技术细节
KVSculpt 将 KV 压缩重新定义为一个**蒸馏（Distillation）问题**，目标是优化一组不受限的 $k$ 个 KV 对，使其产生的注意力输出与原始 $N$ 个 KV 对一致。
*   **两阶段优化策略**：
    *   **Key 优化（L-BFGS）**：由于注意力机制中的 Softmax 导致损失函数非凸且具有尖锐波谷，传统的 Adam 优化器容易陷入局部最优。作者采用二阶拟牛顿法 L-BFGS 来优化 Key 向量。
    *   **Value 求解（最小二乘法）**：在 Key 固定（即注意力权重固定）的情况下，最优的 Value 具有闭式解。作者通过岭回归（Ridge Regression）每隔 5 步 L-BFGS 更新一次 Value。
*   **损失函数**：包含两部分：1) 输出均方误差（MSE），确保注意力加权后的结果一致；2) Log-Sum-Exp (LSE) 匹配，确保该块在后续与新生成的 Token 组合时，其注意力权重比例正确。
*   **合成未来查询（Synthetic Future Queries）**：由于压缩时不知道未来的 Query，作者通过对上下文内容向量进行去旋转（De-RoPE）、均匀采样并重新应用未来位置的 RoPE，构造合成 Query 来增强泛化性。
*   **自适应预算分配**：引入“试点运行”（Pilot Run）机制。先进行短时间的统一压缩，根据各层和各 KV 头的 MSE（压缩难度）动态分配 $k$ 的数量。

### 3. 实验设计
*   **模型与数据**：使用 **Qwen2.5-1.5B-Instruct**。测试数据来自 **PG19** 测试集，上下文长度 $N=2048$，评估后续 128 个 Token 的生成质量。
*   **评估指标**：KL 散度（衡量压缩模型与原始模型输出分布的差异）。
*   **对比基准（Baselines）**：

    *   **Attention Score**：基于注意力分数累加的逐出（类似 H2O）。

    *   **Joint Optimization**：使用硬混凝土松弛（Hard Concrete）学习离散选择门控，是 KVSculpt 的离散对应版本。

*   **实验规模**：主要对比实验在 20 个序列上进行，详细的消融分析和自适应分配实验在 5 个代表性序列上深入展开。

### 6. 主要结论与发现
*   **性能飞跃**：KVSculpt 在相同压缩率下，KL 散度比最强基准 Select+Fit 降低了 **3.5–4.1 倍**。
*   **连续空间的优势**：仅仅优化 Value（Joint Opt）提升有限，允许 Key 在连续空间自由移动是性能提升的关键。

*   **数学严谨**：结合了二阶优化（L-BFGS）和闭式解（最小二乘），充分利用了注意力机制的数学结构。
*   **细粒度控制**：首次深入探讨了 KV 头级别的压缩难度差异，并提出了有效的自适应分配方案。
