## 摘要
任务完成率是衡量大语言模型（LLM）智能体能力的标准指标，但具有相同完成分数的模型在追踪中间状态的能力上可能存在显著差异。我们引入了“工作记忆忠实度-主动操纵”（WMF-AM），这是一种经过校准的、无需草稿纸的累积算术状态追踪探测方法，并在 20 个开源权重模型（参数量 0.5B-35B，涵盖 13 个系列）上针对已发布的确定性 10 任务智能体测试集进行了评估。在预先指定的、经过 Bonferroni 校正的分析中，WMF-AM 预测智能体性能的 Kendall's tau 系数为 0.612（p < 0.001，95% 置信区间 [0.360, 0.814]）；探索性偏 tau 分析表明，在控制完成分数和模型规模后，这一信号依然存在。三项构念隔离消融实验（K=1 对照、非算术上限、配对抵消）支持了以下解释：负载下的累积状态追踪，而非单纯的单步算术或实体追踪，是主要的困难来源。K-校准使探测保持在具有区分度的范围内，而先前的固定深度基准在此时会失去区分度；该结论在开源权重样本之外的泛化性仍有待研究。

## Abstract
Task-completion rate is the standard proxy for LLM agent capability, but models with identical completion scores can differ substantially in their ability to track intermediate state. We introduce Working Memory Fidelity-Active Manipulation (WMF-AM), a calibrated no-scratchpad probe of cumulative arithmetic state tracking, and evaluate it on 20 open-weight models (0.5B-35B, 13 families) against a released deterministic 10-task agent battery. In a pre-specified, Bonferroni-corrected analysis, WMF-AM predicts agent performance with Kendall's tau = 0.612 (p < 0.001, 95% CI [0.360, 0.814]); exploratory partial-tau analyses suggest this signal persists after controlling for completion score and model scale. Three construct-isolation ablations (K = 1 control, non-arithmetic ceiling, yoked cancellation) support the interpretation that cumulative state tracking under load, rather than single-step arithmetic or entity tracking alone, is the primary difficulty source. K-calibration keeps the probe in a discriminative range where prior fixed-depth benchmarks become non-discriminative; generalization beyond this open-weight sample remains open.

### 1. 核心问题与整体含义
*   **研究动机**：当前评估大语言模型（LLM）智能体能力的通用指标是“任务完成率”（Task-completion rate）。然而，作者发现这会导致“完成谬误”（Completion Fallacy）——即两个完成率得分相似的模型，在处理中间步骤的认知能力（如状态追踪）上可能存在巨大差异。
*   **核心问题**：是否存在一种比最终完成率更敏感的“过程探测”指标，能够更准确地预测模型在复杂、多步智能体任务中的表现？
*   **背景**：随着智能体任务步数增加，模型必须在内部维护并更新状态。现有的基准测试往往只看结果，不看模型在负载下维持中间信息的能力。

### 2. 方法论
*   **核心思想**：引入 **WMF-AM（Working Memory Fidelity-Active Manipulation，工作记忆保真度-主动操纵）** 探测器。它借鉴认知科学中“工作记忆”的概念，测试模型在不使用草稿本（No-scratchpad）的情况下，在内部累积更新状态的能力。
*   **关键技术细节**：
    *   **任务设计**：给定一个初始数值状态，随后给出 $K$ 个连续的加减操作指令（如“Alice 得到 5 分”，“Alice 失去 3 分”），要求模型输出最终结果。
    *   **K-校准（K-calibration）**：这是本文的关键创新。通过调整操作步数 $K$（实验中取 3, 5, 7 等），确保探测器处于具有区分度的范围内，避免模型得分全部堆积在 0 或 1。
    *   **CEF 框架**：将 WMF-AM 置于更广泛的“认知评估框架”（Cognitive Evaluation Framework）中，该框架还包含元认知（MCC）、情境记忆（EMC）和认知负荷适应（CLA）等维度。

    *   **对比预测因子**：结果准确率（Outcome Correctness, OC，基于 100 题通用知识测试）、模型参数量（log params）。
*   **对比方法与消融**：
    *   **K=1 对照**：测试单步算术能力，证明模型失败是因为“累积负载”而非“不会算术”。
    *   **非算术上限测试**：将数值改为颜色或位置追踪，测试通用实体追踪能力。
    *   **配对抵消控制（Yoked Cancellation）**：操作序列最终相互抵消回到原点，测试纯算术解析能力。

*   **实验规模**：对 20 个模型进行了多阶段评估，每个 WMF-AM 探测使用 4 个独立随机种子，并测试了 3 种提示词模板（Bare, Chat, CoT）。

### 6. 主要结论与发现
*   **强预测相关性**：WMF-AM 得分与智能体下游性能高度相关（$\tau = 0.612, p < 0.001$），显著优于传统的完成率指标。
*   **增量效用**：在控制了模型规模和通用准确率后，WMF-AM 依然具有显著的预测能力，说明它捕捉到了传统指标忽略的“过程质量”。
*   **认知瓶颈**：实验证明，模型在智能体任务中的失败往往源于在高负载下无法维持累积状态，而非缺乏基础知识或单步逻辑能力。
*   **K-校准的必要性**：证明了像 Minerva 这样固定深度（K=200）的基准会导致所有测试模型得分为 0，从而失去评价意义。

### 7. 优点
*   **诊断深度**：不仅能告诉我们模型“行不行”，还能通过深度降级曲线解释模型“为什么不行”。
*   **方法严密**：通过“配对抵消”和“K=1”等消融手段，极好地完成了构念隔离（Construct Isolation），确保测量的确实是工作记忆。
*   **开源贡献**：发布了确定性的智能体测试集和探测脚本，具有很高的复现价值。

### 8. 不足与局限
*   **样本偏差**：实验对象仅限于 35B 以下的开源模型，尚未在 GPT-4o 或 Claude 3.5 等超大规模闭源 API 模型上验证。
*   **任务局限**：WMF-AM 目前主要基于算术状态，虽然有效，但可能无法完全代表所有类型的智能体认知需求（如复杂的社会协作）。
*   **统计局限**：样本量（N=20）虽然足以进行相关性分析，但不足以进行因子分析（Factor Analysis）来验证其提出的 CEF 四维框架的结构效度。
*   **语言限制**：仅在英文环境下进行了测试。
