## 摘要
大语言模型（LLMs）将海量的世界知识内化为参数化记忆，但不可避免地继承了其源语料库的陈旧性和错误。因此，确保这些内部表示的可靠性和可塑性对于实现可信的现实世界部署至关重要。知识编辑提供了一种无需重新训练即可对记忆进行外科手术式修改的关键范式。然而，尽管近期的编辑器在标准基准测试中表现出极高的成功率，但目前依赖于特定提示条件下评估输出的评估框架是否能可靠地验证真实的记忆修改，仍然存疑。在这项工作中，我们引入了一个简单的诊断框架，使模型在更贴近现实应用环境的上下文学习（ICL）设置下进行判别性自我评估，旨在专门审查由记忆修改引起的细微行为差异。这种探测揭示了一种普遍存在的“表面顺从”（Surface Compliance）现象，即编辑器仅通过模仿目标输出就获得了很高的基准分数，而没有在结构上重写内部信念。此外，我们发现递归修改会积累表示残留，引发认知不稳定，并永久性地降低模型记忆状态的可逆性。这些见解强调了当前编辑范式的风险，并突出了稳健的记忆修改在构建可信、长期可持续的 LLM 系统中的关键作用。代码可在 https://github.com/XiaojieGu/SA-MCQ 获取。

## Abstract
Large Language Models (LLMs) internalize vast world knowledge as parametric memory, yet inevitably inherit the staleness and errors of their source corpora. Consequently, ensuring the reliability and malleability of these internal representations is imperative for trustworthy real-world deployment. Knowledge editing offers a pivotal paradigm for surgically modifying memory without retraining. However, while recent editors demonstrate high success rates on standard benchmarks, it remains questionable whether current evaluation frameworks that rely on assessing output under specific prompting conditions can reliably authenticate genuine memory modification. In this work, we introduce a simple diagnostic framework that subjects models to discriminative self-assessment under in-context learning (ICL) settings that better reflect real-world application environments, specifically designed to scrutinize the subtle behavioral nuances induced by memory modifications. This probing reveals a pervasive phenomenon of Surface Compliance, where editors achieve high benchmark scores by merely mimicking target outputs without structurally overwriting internal beliefs. Moreover, we find that recursive modifications accumulate representational residues, triggering cognitive instability and permanently diminishing the reversibility of the model's memory state. These insights underscore the risks of current editing paradigms and highlight the pivotal role of robust memory modification in building trustworthy, long-term sustainable LLM systems. Code is available at https://github.com/XiaojieGu/SA-MCQ.

### 1. 核心问题与研究动机
*   **核心问题**：当前的知识编辑技术虽然在标准基准测试（如准确匹配率）上取得了极高分数，但这种“成功”是否代表模型真正修改了内部参数化记忆，还是仅仅在输出层面表现出一种**“表面顺从”（Surface Compliance）**？
*   **研究动机**：LLM 的内部记忆具有静态性和错误继承性。知识编辑旨在外科手术式地更新特定事实，但现有评估框架（如 Exact Match）过于依赖生成式输出，容易受到上下文提示的引导，无法探测模型深层的信念改变。作者希望揭示编辑后模型内部记忆的真实状态及其在复杂环境下的稳定性。

### 2. 方法论
论文提出了 **SA-MCQ（Self-Assessment Multiple Choice Question）** 诊断框架，核心思想是将生成任务转变为判别任务，通过在冲突语境下让模型进行“自我评估”来探测其真实信念。
*   **判别性测试**：提供原始事实（Parametric Answer）和编辑后事实（Golden Answer）作为选项，迫使模型在两者间做出选择。
    *   **模式**：包括“二选一”（强制竞争）和“三选一”（加入“不确定”选项）。
*   **似然边际（Likelihood Margin）**：通过计算目标答案与原始答案之间的对数似然差值，量化概率分布的偏移。
*   **外部证据刺激**：设计了四种类型的上下文证据来模拟真实应用：
    *   **参数化证据 (PE)**：支持原始错误记忆的文本。
    *   **黄金证据 (GE)**：支持编辑目标事实的文本。
    *   **无关证据 (IE)**：随机噪声文本。
    *   **反面证据 (CE)**：与原始和目标事实均冲突的虚假文本。
*   **逻辑蕴含检查**：利用 NLI 模型（DeBERTa）确保生成的证据与答案之间存在严格的逻辑支撑关系。

### 3. 实验设计
*   **数据集**：主要使用 **ZsRE**（零样本关系抽取）数据集，因其包含丰富的反事实标注。
*   **基准模型**：LLaMA-3-8B-Instruct 和 Qwen2.5-7B-Instruct。
*   **对比方法**：选取了三种主流编辑器：**AlphaEdit**（空空间约束更新）、**RLEdit**（强化学习驱动）和 **UltraEdit**（轻量级控制器）。
*   **评估指标**：除了传统的有效性（Eff.）、泛化性（Gen.）和特异性（Spe.），重点关注 SA-MCQ 下的黄金答案选择率。

### 4. 资源与算力
*   **硬件设备**：实验在一台 **NVIDIA A800 GPU** 上完成。
*   **算力细节**：论文提到 RLEdit 等方法对显存要求极高（8B 模型需约 79.86 GB VRAM）。评估时间方面，SA-MCQ 非常高效（0.88 分钟），远快于 LLM-as-judge（约 651 分钟）。

### 5. 实验数量与充分性
*   **实验规模**：构建了 1,000 组包含四种证据类型的测试集。
*   **多维度验证**：
    *   进行了**三轮递归编辑实验**（事实 -> 反事实 -> 回归事实），探测记忆的可塑性。
    *   在 MMLU、GLUE 等通用基准上测试了编辑对模型基础能力的副作用。
    *   进行了选项顺序敏感性分析，确保结果不受位置偏见影响。
*   **充分性评价**：实验设计较为全面，涵盖了单次编辑、连续编辑、噪声干扰及多模型验证，逻辑链条完整，对比实验具有说服力。

### 6. 主要结论与发现
*   **表面顺从现象普遍存在**：许多编辑器在生成测试中表现完美，但在 SA-MCQ 判别测试中会原形毕露，倾向于选回原始答案。
*   **编辑后的记忆极其脆弱**：模型对外部上下文高度敏感。即使编辑“成功”，一旦出现支持原始记忆的证据，模型会迅速倒戈。
*   **递归编辑导致“表征残留”**：反复修改同一知识点会积累内部冲突，导致模型进入认知不稳定状态（不确定选项比例激增），且记忆的可逆性随编辑次数增加而永久下降。
*   **副作用显著**：某些编辑器（如 AlphaEdit）虽然修改效果较强，但会显著破坏模型在通用任务（如 MMLU）上的表现。

### 7. 优点
*   **视角独特**：首次系统性地定义并量化了知识编辑中的“表面顺从”风险。
*   **诊断工具实用**：SA-MCQ 框架简单、高效且比传统生成式指标更严苛，能有效识别“伪成功”的编辑。
*   **揭示了长期演化的风险**：通过多轮编辑实验，指出了知识编辑在长期维护 LLM 系统时可能面临的结构性崩溃问题。

### 8. 不足与局限
*   **模型规模限制**：受限于算力（尤其是 RLEdit 的高显存需求），实验仅覆盖了 7B/8B 规模的模型，未在 70B 或更大规模模型上验证结论。
*   **数据集单一**：主要依赖 ZsRE 数据集，虽然该数据集很经典，但对于更复杂、更隐蔽的知识类型（如多跳推理知识）覆盖不足。
*   **应用场景局限**：主要关注事实性知识的修改，未探讨逻辑规则或道德准则编辑中是否存在类似的表面顺从现象。
